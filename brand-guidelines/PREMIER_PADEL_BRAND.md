# Premier Padel Academy Brand Guidelines

## Overview

This document defines the visual identity for Premier Padel Academy. Reference these standards when creating new assets or UI components to ensure consistency.

## Identity

**Core Values:** Excelencia, Dedicación, Esfuerzo.

## Visual Style

### Color Palette

#### Primary Colors
- **Empire Green:** `#2d4a3e`
  - Used for: Headers, primary buttons, team stats, borders.
  - Context: Represents the court, professionalism, and stability.

- **Golden Trophy:** `#daa520`
  - Used for: Accents, borders, highlights, active states.
  - Context: Represents victory, prestige, and quality.

#### Secondary Colors
- **Victory Green:** `#4CAF50` (Success states, "Victoria" badges)
- **Defeat Red:** `#f44336` (Error states, "Derrota" badges)
- **Deep Forest:** `#1a2e20` (Gradient end-points)

#### Neutrals
- **Glass White:** `rgba(255, 255, 255, 0.9)`
  - Used for: Cards, sections, backdrops.
- **Text Gray:** `#333333` (Primary text), `#666666` (Secondary text).

### Typography

- **Font Family:** `'Segoe UI', Tahoma, Geneva, Verdana, sans-serif`
- **Headings:** Bold, often with gradients or border-bottom accents.
- **Body:** Clean, legible, high contrast against white backgrounds.

### UI Components

#### Glassmorphism
The UI heavily utilizes glassmorphism effects:
```css
background: rgba(255, 255, 255, 0.9);
backdrop-filter: blur(10px);
box-shadow: 0 10px 30px rgba(0, 0, 0, 0.1);
border-radius: 20px;
```

#### Buttons & Interactives
- **Primary:** Gradient Green to lighter Green.
- **Hover Effects:** `transform: translateY(-3px)` and increased shadow.
- **Borders:** Often 2px solid `#daa520`.

### Logo
- **Icon:** A shield emblem containing padel rackets and ball.
- **Composition:** Gold outlines on a Green shield.
